# Open Universe (Hayverse)

Нодовая среда открытой киновселенной армянского мира — UGC-платформа «Weavy × Git».
Пользователи собирают сцены из нод-объектов (персонаж, локация, одежда армянских
дизайнеров, здание, мебель, искусство, транспорт, музыка, сценарий, раскадровка),
а версии канонизируются через git-подобный процесс (форк → pull-request →
модерация → merge в канон); непринятое живёт в мультивселенной. Из сцен
собирается фильм, который можно смотреть с живыми регуляторами или в любой момент
переключить в игру.

> Это самостоятельный проект, не связанный с PROUN-игрой.

## Прототипы

Автономные HTML-файлы (без сборки, открываются двойным кликом; иконки — Tabler с
CDN, тема светлая/тёмная):

| Файл | Что демонстрирует |
|---|---|
| [`prototypes/index.html`](prototypes/index.html) | **Актуальный** — сшитый экран: граф слева → плеер справа, изменение узла мгновенно перерисовывает кадр. |
| [`prototypes/prototype-1-node-editor.html`](prototypes/prototype-1-node-editor.html) | Нодовый редактор: граф, инспектор, Канон/Мультивселенная, очередь pull-request, таймлайн. |
| [`prototypes/prototype-2-film-game-player.html`](prototypes/prototype-2-film-game-player.html) | Плеер «фильм ↔ игра»: живые регуляторы и вход в игру с любого таймкода. |

## Запуск / превью

Файлы статические — можно открыть двойным кликом. Либо локальный сервер:

```
python -m http.server 4174
```

и открыть `http://localhost:4174/prototypes/index.html`.
В Claude Code предусмотрен конфиг превью — `.claude/launch.json` (сервер `openuniverse`).
