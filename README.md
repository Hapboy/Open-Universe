# Open Universe (Hayverse)

Нодовая среда открытой киновселенной армянского мира — UGC-платформа «Weavy × Git».
Пользователи собирают сцены из нод-объектов (персонаж, локация, одежда армянских
дизайнеров, здание, мебель, искусство, транспорт, музыка, сценарий, раскадровка),
а версии канонизируются через git-подобный процесс (форк → pull-request →
модерация → merge в канон); непринятое живёт в мультивселенной. Из сцен
собирается фильм, который можно смотреть с живыми регуляторами или в любой момент
переключить в игру.

> Это самостоятельный проект, не связанный с PROUN-игрой.

## Приложение (PWA) — `app/`

Устанавливаемая нодовая студия (offline-capable PWA, service worker + manifest):

- **Нодовый редактор** с типизированными портами — выход ноды соединяется только
  со совместимым входом (Image / Video / Text / Audio).
- **Ноды Higgsfield AI:** Soul (генерация кадра), Motion (камера-мув), Speak
  (липсинк) — + Pinterest, текст, выходная сцена.
- **Pinterest-нода:** подключается к доске, выбираемой в выпадающем списке, и
  отдаёт пины как референсы.
- **Мультиплеер-разработка:** каждый регистрируется, создаёт персонажа, выбирает
  сторону и роль по сценарию. **В финальной сцене в кадр выходят все разработчики**
  вместе с вымышленными персонажами.
- **Генеративный плеер** сцен встроен в инспектор.

API работают в **mock-режиме без ключей** (демо-результаты) и переключаются в
боевой режим, когда в Настройках введён ключ Higgsfield / токен Pinterest (ключи
хранятся только в браузере). ⚠️ Эндпоинты Higgsfield в адаптере иллюстративны —
выровнять по реальной документации перед боевым запуском.

Запуск: `python -m http.server 4174` в корне проекта, открыть
`http://localhost:4174/app/index.html` (или открыть `app/index.html` напрямую).
